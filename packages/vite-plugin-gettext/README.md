# vite-plugin-gettext

A Vite plugin for compiling Gettext PO files (`.po`) to MO files.


## Installation

```bash
npm install @easy6502/vite-plugin-gettext --save-dev
```

or if you use Yarn:

```bash
yarn add @easy6502/vite-plugin-gettext --dev
```
